Colour Switch

This project is a simple colour switcher. The premise is to provide a button which will change the hex colour at random. In order to do this, I needed to use the built in Math.random and Math.floor functions to loop through an array of the 6 HEX digits.

Number Counter

Number Counter uses 3 buttons - reset, decrease and increase. An if/else statement allows the buttons to increment or decrement each time a 'click' event occurs on which ever button. Thus, the amount on the screen changes each time.

Photo Carousel

This simple carousel of photos uses 3 buttons - previous, next or random. The concept is that each time a button is clicked, a related function will be triggered to increment or decrement the id of the photo. This then also accesses the showPhoto function, to allow the other linked information to change each time with the photo.

Responsive Nav

The responsive nav-bar uses flexbox tools to hide and access the links within the nav, so that it can then push them below in column form. This is useful to save space and is allowed to happen through the use of the button which appears when the screen is below 450px in width.
